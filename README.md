# Project Aegis Ghost - Complete Documentation Index

## Overview
Project Aegis Ghost is an advanced encryption and steganography system with multi-factor biometric authentication, now featuring AI-powered gesture challenges using Google Gemini AI.

## Core System Documentation

### Main Execution
- **[main1.py](./main1.py)** - Entry point for lockdown and recovery phases
  - `python main1.py` - Standard encryption/steganography
  - `python main1.py --biometric` - With multi-factor authentication

### Core Modules

1. **[core/steganography.py](./core/steganography.py)**
   - DWT-based data embedding and extraction
   - Lossless coefficient storage (.npy files)
   - `embed_data_dwt()` and `extract_data_dwt()` functions

2. **[core/encryption.py](./core/encryption.py)**
   - AES-GCM encryption
   - Shamir Secret Sharing (6-of-10 threshold)
   - `encrypt_and_shatter()` and `biometric_unlock()` functions

3. **[core/ai_engine.py](./core/ai_engine.py)**
   - AI-driven carrier image generation
   - Biometric authentication integration
   - `generate_ghost_carrier()` and `verify_owner()` functions

4. **[core/biometric_auth.py](./core/biometric_auth.py)**
   - Face recognition verification
   - Voice authentication (Google Speech Recognition)
   - Gesture challenge verification
   - Multi-factor authentication orchestration

5. **[core/gesture_auth.py](./core/gesture_auth.py)** ⭐ **NEW**
   - AI-powered gesture challenge generation (Gemini AI)
   - Real-time face detection (OpenCV cascade)
   - Head pose estimation (yaw, pitch, roll)
   - Gesture detection and verification

## System Overview

### Architecture Flow

```
LOCKDOWN (Hide)
├─ Read secret data
├─ Encrypt with AES-GCM
├─ Split key with Shamir Secret Sharing
├─ Generate carrier image
├─ Embed shares via DWT steganography
└─ Output protected carrier image

RECOVERY (Reveal) with Biometric Auth
├─ Extract steganographic shares
├─ Verify owner via:
│  ├─ Face Recognition (compare with assets/owner.jpg)
│  ├─ Gesture Challenge (AI-generated by Gemini)
│  └─ Voice Authentication (speech recognition)
├─ Reconstruct encryption key from shares
├─ Decrypt secret data
└─ Output original secret
```

### Technology Stack

| Component | Technology | Purpose |
|-----------|-----------|---------|
| Encryption | PyCryptodome (AES-GCM) | Military-grade data protection |
| Key Sharing | Shamir Secret Sharing | Distributed trust model |
| Steganography | DWT (Discrete Wavelet Transform) | Lossless data hiding |
| Face Recognition | face_recognition library | Owner verification |
| Voice Auth | SpeechRecognition + Google API | Voice command recognition |
| Gesture Auth | OpenCV + Gemini AI | AI-powered challenge generation |
| Carrier Images | Stable Diffusion / Mock | Image generation |

## Common Commands

```bash
# Install all dependencies
pip install -r core/requirements.txt

# Standard encryption/steganography (no auth)
python main1.py

# With biometric authentication (face + gesture + voice)
python main1.py --biometric

# Test gesture generation
python -c "from core.gesture_auth import GestureAuthenticator; auth = GestureAuthenticator(); print(auth.generate_gesture_challenge())"

# Test face verification
python -c "from core.biometric_auth import BiometricAuthenticator; auth = BiometricAuthenticator(); print(auth.verify_face(timeout=5))"

# Test voice verification
python -c "from core.biometric_auth import BiometricAuthenticator; auth = BiometricAuthenticator(); print(auth.verify_voice(timeout=5, phrase='authenticate'))"
```

## Advanced Configuration

### Adjust Gesture Sensitivity
Edit `core/gesture_auth.py` - `analyze_movement()` method:
```python
if yaw_change < -0.4:  # Change -0.4 to adjust sensitivity
    return "look_left"
```

### Increase Authentication Timeout
Edit `core/gesture_auth.py` - `verify_gesture()` method:
```python
success, msg = auth.verify_gesture(capture_duration=10)  # Increased from 5
```

### Disable Face/Voice/Gesture
Edit authentication calls:
```python
auth.authenticate(
    enable_face=False,    # Disable face recognition
    enable_gesture=False, # Disable gesture
    enable_voice=False    # Disable voice
)
```
